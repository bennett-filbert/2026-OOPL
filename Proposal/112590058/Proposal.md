# Game Introduction

This project is a 2D tower-defense game inspired by Plants vs. Zombies.

The player places plants on a grid to defend against zombies walking from the right side of the screen.  
Plants automatically attack zombies, and the player must stop zombies from reaching the house.

The goal of the game is to survive several waves of zombies using a small set of plants with different abilities.
