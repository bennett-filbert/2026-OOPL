# Game Introduction

遊戲中，玩家操作主角馬力歐攀爬架構、躲避障礙，從大猩猩大金剛手中救出碧姬公主。此作不僅是馬力歐的初登場，更奠定了橫向平台遊戲的基礎。

此遊戲包含四種地圖，多個關卡，每輪三條生命值。
