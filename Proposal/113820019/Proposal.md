# Game Introduction

《植物大戰殭屍》是一款經典的塔防益智遊戲。玩家需在自家草坪上種植多種具備特殊攻擊或防禦能力的植物（如豌豆射手、堅果牆），以抵禦一波波滑稽且各具特色的殭屍入侵。遊戲節奏明快，策略性十足，結合了資源管理（收集陽光）與多樣化的關卡設計（白天、夜晚、泳池等），深受各年齡層玩家喜愛。

遊戲影片 : https://www.youtube.com/watch?v=XENla8M3910
